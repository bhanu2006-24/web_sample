# Aurum Fine Jewellery (Krishna Jewellery)

A luxury jewellery store website featuring a sophisticated dark and gold aesthetic, designed to showcase high-end collections.

## üíé Features

- **Luxury Aesthetic**: A premium design using deep black (`#1a1a1a`) and muted gold (`#cfa05d`) colors, paired with serif typography (`Cormorant Garamond`).
- **Single Page Application (SPA)**: Seamless navigation between Home, Collections, Product Details, Cart, and Checkout.
- **Dynamic Product Catalog**: A simulated catalog of 50+ items with filtering by category (Rings, Necklaces, Earrings, Bracelets).
- **Detailed Product Views**: Rich product pages with technical specifications (Metal, Carat, Clarity, Cut).
- **Shopping Cart & Checkout**: Functional cart system with a multi-step checkout simulation.
- **Firebase Integration**: Setup for Firebase Auth and Firestore (runs in standalone mode if not configured).
- **Client Care Pages**: Dedicated sections for Shipping, Warranty, Returns, and Contact.
